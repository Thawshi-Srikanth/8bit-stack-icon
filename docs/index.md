A collection of 8-bit / pixel-art style icons for popular tech stack technologies.

## Features

- **Multi-Framework**: React, Vue, Svelte supported.
- **Pixel Perfect**: Designed on a 16x16 grid (mostly).
- **TypeScript**: Fully typed.
