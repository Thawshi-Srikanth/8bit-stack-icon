# 8-bit Stack Icons (Monorepo)

A library of 8-bit / pixel-art style icons for popular tech stack technologies.
Now supporting multiple frameworks!


## Installation

```bash
npm install @8bit-stack-icon/icons
```

## Usage

This package contains the raw SVGs.

```javascript
import stack from '@8bit-stack-icon/icons/base/stack.svg';
```
