# 8-bit Stack Icons (Monorepo)

A library of 8-bit / pixel-art style icons for popular tech stack technologies.
Now supporting multiple frameworks!


## Installation

```bash
npm install @8bit-stack-icon/vue
# or
yarn add @8bit-stack-icon/vue
# or
pnpm add @8bit-stack-icon/vue
# or
bun add @8bit-stack-icon/vue
```

## Usage

```vue
<script setup>
import { StackIcon } from '@8bit-stack-icon/vue';
</script>

<template>
  <StackIcon name="vue" :size="32" />
</template>
```
